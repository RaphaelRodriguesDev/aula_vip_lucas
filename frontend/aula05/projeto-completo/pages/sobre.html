Objetivo do Projeto

Este projeto foi desenvolvido para demonstrar a aplicação prática de todos os conceitos aprendidos nas aulas de HTML e CSS, incluindo:

  • HTML Semântico: Uso correto de tags semânticas como header, nav, main, section, article, aside e footer
  • Formulários: Implementação de formulários completos com validação
  • Tabelas: Estruturação de dados tabulares com rowspan e colspan
  • CSS Moderno: Flexbox, Grid, gradientes e animações
  • Responsividade: Design adaptável para diferentes dispositivos

Tecnologias Utilizadas

Tecnologia Descrição Aplicação no Projeto
